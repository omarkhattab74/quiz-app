🧠 Quiz App

An interactive Quiz Application built with JavaScript, HTML, and CSS that allows users to customize their quiz experience by selecting the number of questions, category, difficulty level, and question type.

The application fetches questions dynamically and provides a timed quiz experience, displaying the final score once the quiz is completed.

🚀 Features

🎯 Custom Quiz Setup

Select the number of questions

Choose question category

Select difficulty level (Easy / Medium / Hard)

Choose question type (Multiple Choice / True or False)

⏱ Quiz Timer

Each quiz has a time limit to answer the questions.

❓ Interactive Questions

Dynamic question rendering

Multiple-choice answer selection

📊 Score Calculation

Shows the user's final score after finishing the quiz.

🔄 Restart Quiz

Users can start a new quiz anytime.

📱 Responsive UI

Works smoothly on different screen sizes.

🛠 Tech Stack

JavaScript (Vanilla JS)

HTML5

CSS3

Open Trivia Database API (for quiz questions)

▶️ How the Application Works

The user selects:

Number of questions

Category

Difficulty level

Question type

The application sends a request to the Open Trivia API to retrieve quiz questions.

Questions are displayed one by one with answer options.

The quiz includes a timer, and users must answer within the given time.

After completing the quiz, the application displays the final score.
